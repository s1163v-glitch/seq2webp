# Seq2WebP — 이미지 시퀀스 → 애니메이션 WebP 변환기

## 실행 전 준비

1. **Node.js** 설치 (https://nodejs.org — LTS 버전 권장)
2. 이 폴더를 원하는 위치에 두세요

## 처음 실행 (최초 1회만)

```
npm install
```

> sharp 패키지가 네이티브 바이너리를 다운로드하므로 인터넷 연결이 필요합니다.
> 수 분 이내에 완료됩니다.

## 실행

```
npm start
```

## 빌드 (설치 필요 없는 .exe 만들기)

```
npm run build
```

완료되면 `dist/` 폴더에 `Seq2WebP.exe` 가 생성됩니다.

## 지원 포맷

입력: PNG, JPEG, TIF/TIFF, WebP, BMP  
출력: 애니메이션 WebP

## 기능

- 파일 드래그&드롭 또는 클릭하여 선택
- FPS, 루프 횟수, 화질 설정
- 출력 해상도 지정 (너비/높이 입력, 비워두면 원본 크기)
- 변환 후 미리보기
- 저장 경로 선택
